# Minimum-Cost-Flow
A simple application to demonstrate Minimum-Cost Flow problem on graph using two algorithms:
* Cycle canceling
* Minimum mean cycle canceling

_Built in **Java SE 15**, graphic using **JavaFX SDK 16** and IDE **eclipse**_
